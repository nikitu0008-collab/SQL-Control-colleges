# 🗄️ SQL College Labs

[![SQL](https://img.shields.io/badge/SQL-PostgreSQL%20%7C%20MySQL-316192?logo=postgresql)]()

> Решения контрольных и своих работ по курсу «Базы данных» (колледж).  
> Код написан в учебных/практических целях и демонстрирует владение языком SQL на уровне сложных/лёгких запросов, проектирования схем и оптимизации.

---

## 🛠️ Используемые технологии

- **СУБД**: PostgreSQL 15 / MySQL 8.0 (указано в начале каждого файла)
- **Инструменты**: DBeaver, pgAdmin, MySQL Workbench
  
---

## 🚀 Как запустить

1. Установите PostgreSQL или MySQL (согласно требованиям конкретного скрипта).
2. Создайте новую базу данных.
3. Выполните скрипт в вашем SQL-клиенте:
   ```bash
   psql -U username -d database_name -f lab1_schema.sql

или скопируйте содержимое в редактор SQL и выполните по частям.

    ⚠️ Некоторые файлы содержат тестовые данные (INSERT). Они предназначены только для проверки работоспособности.

🧠 О чём эти работы

В этом репозитории представлены контрольные работы, которые я выполнял в колледже в рамках курса по базам данных.
Также здесь будут появляться решения задач с платформы SQL Academy — они помогают оттачивать навыки написания запросов на реальных примерах.

В ходе выполнения всех этих заданий были закреплены следующие навыки:

    Проектирование реляционных схем (1НФ–3НФ)

    Создание сложных SELECT с JOIN, подзапросами и WITH

    Использование оконных функций (ROW_NUMBER, RANK, LAG/LEAD)

    Написание хранимых процедур, функций и триггеров

    Работа с представлениями (VIEW)

    Оптимизация запросов через индексы и анализ плана выполнения

🔗 Связь с C++

Являясь C++ разработчиком, я считаю, что понимание SQL необходимо для создания высокопроизводительных серверных приложений.
Этот репозиторий дополняет моё основное портфолио на C++, показывая, что я:

    могу проектировать эффективные схемы данных

    умею писать сложные запросы, которые лягут в основу backend-сервисов

    понимаю, как правильно взаимодействовать с БД из C++ (через ODBC, libpq, etc.)

⚠️ Важно об учебных работах

Данный репозиторий создан для портфолио и демонстрации навыков.
Если вы студент: не копируйте код без понимания. Лучше разберитесь в логике и напишите свою реализацию — это единственный способ действительно научиться.
